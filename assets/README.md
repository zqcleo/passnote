# 资源文件说明

此目录用于存放应用图标和启动画面。

## 需要的文件

1. **icon.png** - 应用图标 (1024x1024px)
2. **adaptive-icon.png** - Android自适应图标 (1024x1024px)
3. **splash.png** - 启动画面 (1284x2778px)
4. **favicon.png** - Web图标 (48x48px)

## 临时方案

当前项目可以正常运行，Expo会使用默认图标。

## 自定义图标

你可以：
1. 使用设计工具创建图标
2. 使用在线工具如 https://www.appicon.co/
3. 或使用 Expo 的图标生成工具

将生成的图标文件放入此目录即可。
